# prml-manim

Manim を使って Christopher M. Bishop, *Pattern Recognition and Machine Learning* (2006) の内容を章・節ごとに視覚化し、解説動画として制作するためのプロジェクトです。

## セットアップ

```bash
uv sync
```

## 構成

- `feature/`: 章ディレクトリ配下に節ディレクトリを置く動画制作単位
- `skills/`: このリポジトリで使うローカル Codex skill
- `reports/working/`: 作業完了レポート
- `.working/`: PDF や一時素材置き場。Git 管理対象外
