## Scene 1: 不確かさを数で扱う

**画面**

前回の多項式曲線を思わせる赤い平均曲線を表示し、周囲に青い観測点と薄い不確かさの帯を出す。タイトル「1.2 Probability Theory」を表示する。

**ナレーション**

前回は、データの点から一本の曲線を選びました。
でも実際のデータには、ノイズがあります。
同じ入力 x でも、観測される t は少しずつ揺れます。
だから知りたいのは、答えが一つだけ、という話ではありません。
どの値がどれくらいありそうかを、まとめて表す必要があります。
そのための言葉が、確率です。

**原文参照**

PRML 1.2 冒頭では、不確実性を扱う体系として確率論を導入し、パターン認識における中心的役割を説明している。

## Scene 2: 確率は可能性の重み

**画面**

3 つのクラス `C_1, C_2, C_3` の棒グラフを表示する。各棒の上に確率値を表示し、合計が 1 になることを示す。

**ナレーション**

確率は、起こり方に付ける重みだと思えます。
重みはゼロ以上で、全部を足すと一になります。
この棒グラフでは、どのクラスがどれくらいありそうかを表しています。
一番高い棒だけを見るのではなく、ほかの可能性もまだ残しておきます。
機械学習では、この残り方がとても大事です。

**原文参照**

PRML 1.2.1 は、確率の和と積の法則を導入する前提として、確率が事象の不確実性を表す量であることを説明している。

## Scene 3: 同時確率と周辺化

**画面**

`p(X,Y)` の 3x3 表を表示し、行方向・列方向へ足し合わせるアニメーションで `p(X)` と `p(Y)` を作る。

**ナレーション**

二つの変数を同時に見るときは、表の一マスが一つの組み合わせを表します。
たとえば、クラスと特徴が同時にこうなる、という重みです。
行を全部足すと、そのクラス全体の確率になります。
列を全部足すと、その特徴全体の確率になります。
見たい変数だけを残して、ほかを足し消す。これが周辺化です。

**原文参照**

PRML Eq. (1.5) と Eq. (1.6) は、同時確率から一方の変数について和を取ることで周辺確率を得る和の法則を示している。

## Scene 4: 条件付き確率と積の法則

**画面**

同時確率の表のうち `X=x_2` の列を強調する。列全体の重み `p(X=x_2)` と、その中での割合 `p(Y|X=x_2)` を並べて表示し、積の法則へ接続する。

**ナレーション**

条件付き確率は、条件を聞いたあとの確率です。
X が分かったあとで、クラスがどれくらいありそうかを見る、という形です。
同時確率は、条件付き確率と、条件そのものの確率に分けられます。
式では、p(x,y) は、p(x) かける p(y|x)、と書けます。
全体の重みを、入口の重みと、そこから先の割合に分けているだけです。

**原文参照**

PRML Eq. (1.7) は条件付き確率を導入し、Eq. (1.8) は積の法則 `p(X,Y)=p(Y|X)p(X)` を示している。

## Scene 5: ベイズの定理

**画面**

`prior` の棒、`likelihood` の棒、掛け算後の未正規化棒、`posterior` の棒へ順に変形する。式 `p(C|x)=p(x|C)p(C)/p(x)` を表示する。

**ナレーション**

ベイズの定理は、見方を反対向きにするための道具です。
原因からデータが出る確率が分かっているとします。
でも実際に欲しいのは、データを見たあとで、どの原因がありそうかです。
事前の重みと、データの出やすさを掛け合わせます。
最後に、全部を足して一になるように割り直します。
これが、観測したあとに信念を更新する、ということです。

**原文参照**

PRML Eq. (1.12) は、積の法則を二通りに書くことでベイズの定理を導出している。

## Scene 6: 密度は面積で読む

**画面**

ガウス分布の密度曲線を表示し、区間 `[a,b]` の下を塗る。高さではなく面積を強調する。

**ナレーション**

連続的な値では、一点ちょうどの確率ではなく、区間の確率を考えます。
曲線の高さは確率そのものではありません。
区間の下にある面積が、その範囲に入る確率です。
面積を全部足すと一になるように作られた曲線を、確率密度と呼びます。
ガウス分布は、平均の近くが高く、遠くへ行くほど低くなる密度です。

**原文参照**

PRML 1.2.1 の probability densities では、連続変数に対して区間内に入る確率を密度の積分で表す。

## Scene 7: 期待値と分散

**画面**

離散分布の棒グラフを表示し、期待値を重心の垂直線として示す。次に平均からのズレの矢印を出し、分散の式を表示する。

**ナレーション**

分布を一つの代表値で見るなら、期待値を使います。
期待値は、確率の重みを付けた平均です。
棒が重い場所に、平均の印が引き寄せられます。
一方で、平均だけでは広がり方は分かりません。
平均からどれくらい散らばっているかを見る量が、分散です。
中心と広がりを分けて見ると、不確かさを扱いやすくなります。

**原文参照**

PRML Eq. (1.33) は期待値を、Eq. (1.38) は分散を定義している。

## Scene 8: 回帰を確率モデルにする

**画面**

回帰曲線を平均として表示し、周囲にガウスノイズの帯を重ねる。観測点はその帯の中に散らばる。式 `p(t|x,w,\beta)=N(t|y(x,w),\beta^{-1})` を表示する。

**ナレーション**

確率論を使うと、前回の曲線フィッティングも見方が変わります。
予測曲線は、t の平均を表している、と考えます。
実際の観測値は、その平均のまわりにノイズを持って散らばります。
式では、t は y(x,w) を中心にしたガウス分布から出る、と書けます。
一本の曲線だけでなく、どれくらい不確かかも一緒に持てるようになります。
ここから、尤度、ベイズ推論、確率的なモデル選択へ進んでいきます。

**原文参照**

PRML 1.2 の確率論は、後続の最尤推定、ベイズ推論、モデル選択を支える土台になる。1.1 の曲線フィッティングも、ノイズ付きの確率モデルとして再解釈できる。
