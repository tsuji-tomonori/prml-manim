## Scene 1: 勾配の次に曲率を見る

**画面**

`nabla E(w)` から `H=nabla nabla E(w)` へ矢印を伸ばし、`H_ij = partial^2 E / partial w_i partial w_j` を表示する。下部に「最適化」「刈り込み」「ラプラス近似」を並べる。

**ナレーション**

ニューラルネットワークの訓練では、誤差関数を重みで微分した勾配を使って、重みを動かします。
ヘッセ行列は、その勾配がさらにどう変わるかを集めた、二階微分の行列です。
一つ一つの成分は、重みアイと重みジェイを少し変えたとき、誤差の曲がり方がどう結びつくかを表します。
そのため、単なる傾きではなく、誤差面の細長さ、平坦さ、交差する曲率まで見ることができます。
この節では、ヘッセ行列を、曲率を読む地図として見ていきます。

**原文参照**

PRML p.249 の Eq. (5.78) は、重みについての二階微分を導入し、それらを単一ベクトル表現の `H_ij` としてまとめる。

## Scene 2: 局所二次近似

**画面**

重み空間の等高線、最小点 `w_hat`、現在点 `w`、差分 `w-w_hat` を表示する。右側に局所二次近似と、最小点で一次項が消える式を順に出す。

**ナレーション**

まず、ある重みの近くで誤差関数をテイラー展開します。
一次の項には勾配が入り、二次の項にはヘッセ行列が入ります。
最小点の近くでは勾配がゼロなので、局所的な形は二次の項がほぼ決めます。
つまりヘッセ行列は、最小点の近くの誤差面を、楕円形のボウルとして近似するときの形そのものです。
勾配が場所を動かす量なら、ヘッセ行列はその周りの地形を説明する量です。

**原文参照**

PRML p.237-238 の Eq. (5.28)-(5.32) は、誤差関数の局所二次近似を導入し、最小点では線形項が消えることを示している。

## Scene 3: 固有値と正定値性

**画面**

楕円の等高線に固有方向 `u_1`, `u_2` を重ねる。`H u_i = lambda_i u_i` と `E = E(w_hat) + 1/2 sum lambda_i alpha_i^2` を表示し、正定値性の判定をカード化する。

**ナレーション**

ヘッセ行列を固有値分解すると、固有ベクトルが曲率の主な向きを表します。
対応する固有値が大きい方向では、少し動いただけで誤差が急に増えます。
固有値が小さい方向では、等高線が長く伸び、誤差面は平坦に見えます。
すべての固有値が正なら、その点のまわりではどの方向に動いても誤差が増えるので、局所最小と判断できます。
負の固有値やゼロに近い固有値は、鞍点や平坦な谷を疑うサインになります。

**原文参照**

PRML p.238-239 の Eq. (5.33)-(5.40) は、ヘッセ行列の固有値分解、等高線の楕円、正定値性と局所最小条件を説明している。

## Scene 4: ニューラルネットワークでの役割

**画面**

中央に `H in R^{W x W}` の行列を置き、四方向に「二次の最適化」「再学習の近似」「重みの刈り込み」「ベイズ推論」を配置する。下に計算量の注意を表示する。

**ナレーション**

ヘッセ行列は、ニューラルネットワークの訓練と解析のいくつかの場面で重要です。
二次の最適化では、曲率を使って、どの方向にどれくらい進むかを調整します。
訓練データが少し変わったときの再学習の近似にも使えます。
さらに、逆ヘッセ行列は、重みを削っても影響が小さいかを調べる刈り込みの考え方に使われます。
ベイズニューラルネットワークでは、ラプラス近似、予測分布、モデル証拠にも現れます。
ただし重みが W 個あると、ヘッセ行列は W かける W の大きさになります。

**原文参照**

PRML p.249-250 の 5.4 冒頭は、最適化、再学習、pruning、Bayesian neural network の Laplace approximation におけるヘッセ行列の役割を列挙している。

## Scene 5: 近似の考え方

**画面**

対角近似、外積近似、正確評価の 3 枚のカードを並べる。二乗誤差のヘッセ行列を、一階微分外積の和と残差付き二階微分項に分け、後者を小さいとみなすことを示す。

**ナレーション**

大きなヘッセ行列をそのまま扱うのは重いので、いくつかの近似が使われます。
対角近似では、非対角成分をゼロにして、重み同士の結びつきを捨てます。逆行列は簡単ですが、実際のヘッセ行列は強く非対角なことがあります。
外積近似では、各データ点の一階微分ベクトルの外積を足し合わせます。
これは、二乗誤差で、訓練後の残差が小さいときに使いやすい近似です。
近似を使うときは、計算を軽くする代わりに、どの項を無視しているかを確認します。

**原文参照**

PRML p.250-252 の Eq. (5.79)-(5.85) は、対角近似と outer product approximation を導入し、それぞれの計算量と注意点を説明している。

## Scene 6: 逆ヘッセ行列と有限差分

**画面**

左に `H_L^{-1}` から `H_{L+1}^{-1}` への更新式を置く。右に 4 点の有限差分ステンシルと二階微分近似式を置き、計算量を表示する。

**ナレーション**

外積近似を使うと、データ点を一つずつ足しながら、逆ヘッセ行列を更新できます。
これはウッドベリー恒等式の形で、すでにある逆行列に、新しいデータ点の情報を吸収していきます。
一方、二階微分は有限差分でも計算できます。
ただし、すべての重みの組を直接差分で見ると計算量が大きくなります。
実用上、有限差分は本番の計算方法というより、二階 backprop の実装確認に向いています。

**原文参照**

PRML p.252-253 の Eq. (5.86)-(5.91) は、outer product approximation からの逆ヘッセ行列の逐次更新と、有限差分による二階微分の計算を説明している。

## Scene 7: 正確な評価とヘッセベクトル積

**画面**

ネットワーク図に forward と backward の流れを重ね、`v^T H = v^T nabla(nabla E)`、`R{w}=v`、`H full: O(W^2), Hv: O(W)` を表示する。

**ナレーション**

ヘッセ行列は、backpropagation を二階微分へ拡張することで、正確に評価できます。
ただし、多くの応用では、行列全体ではなく、ヘッセ行列とあるベクトルの積だけが欲しいことがあります。
このときは、R オペレーターを使って、通常の forward と backward に対応する追加の量を流します。
すると、全ヘッセ行列を保存せずに、ヘッセベクトル積を効率よく得られます。
ヘッセ行列は、曲率を読む道具です。必要な精度と計算量に合わせて、近似、正確な評価、ベクトル積を選びます。

**原文参照**

PRML p.253-256 の Eq. (5.92)-(5.111) は、二層ネットワークでの正確なヘッセ行列の評価と、R operator による Hessian-vector product の効率的な計算を説明している。
