## Scene 1: 重みを一点ではなく分布として扱う

**画面**

左に MAP の分類器、右に重み空間の楕円状の事後分布を表示する。下部に `p(w|t) proportional p(w) prod_n p(t_n|w)` を置く。

**ナレーション**

ロジスティック回帰では、重みベクトル w を一つ選ぶと、シグモイド関数でクラス確率が決まります。
ベイズロジスティック回帰では、この w を一点推定ではなく、事後分布として持ちます。
すると、分類境界だけでなく、その境界をどれくらい信じてよいかも表せます。
ただし、事後分布は、事前分布と、データ点ごとのシグモイド尤度の積になります。
この積を正確に正規化することも、予測時に積分することも、解析的には難しくなります。

**原文参照**

PRML p.217 の 4.5 冒頭は、ロジスティック回帰の厳密なベイズ推論では、シグモイド尤度の積と事前分布の積を正規化する必要があり、事後分布と予測分布の評価がどちらも intractable になると述べている。

## Scene 2: 事前分布と尤度から事後分布を作る

**画面**

1 次元の模式図として、ガウス事前分布、シグモイド尤度の積、事後分布を重ねて表示する。下部に Eq. (4.142) のログ事後分布を出す。

**ナレーション**

まず、重み w にガウス事前分布を置きます。
データを観測すると、それぞれの点が、シグモイド尤度として w の値に制約をかけます。
ログ事後分布は、事前分布の二次項と、交差エントロピー型の対数尤度の和になります。
この形は、線形回帰のようなガウス同士の共役な積ではありません。
そこで、事後分布そのものを扱う代わりに、近くでガウスに置き換える準備をします。

**原文参照**

PRML p.217-218 の Eq. (4.140) は `p(w)=N(w|m0,S0)` のガウス事前分布を置く。Eq. (4.141) は `p(w|t) proportional p(w)p(t|w)`、Eq. (4.142) はガウス事前分布の二次項とロジスティック回帰の対数尤度を足したログ事後分布を与える。

## Scene 3: ラプラス近似でMAPの近くをガウスにする

**画面**

事後分布曲線の最大点を `w_MAP` として縦線で示し、その周辺にガウス近似曲線を重ねる。`q(w)=N(w|w_MAP,S_N)` と `S_N^{-1}` の式を表示する。

**ナレーション**

ラプラス近似では、まず事後分布が最大になる点、つまり MAP 解を探します。
次に、その点での曲率をヘッセ行列で測ります。
PRML 4.5.1 では、近似事後分布を、平均 w MAP、共分散 S N のガウス分布として書きます。
曲率が大きい方向は、少し動かすだけで事後確率が下がるので、分散が小さくなります。
曲率が小さい方向は、データだけでは重みをよく決められていない方向です。

**原文参照**

PRML p.218 の Eq. (4.143) は、事後分布の負の二階微分から `S_N^{-1}=S_0^{-1}+sum_n y_n(1-y_n) phi_n phi_n^T` を与える。Eq. (4.144) は `q(w)=N(w|w_MAP,S_N)` のガウス近似を定義する。

## Scene 4: MAPのロジスティック分類器

**画面**

2 次元の自作 2 クラスデータに対して、`sigma(w_MAP^T phi)` の確率面を色で表示する。白い線で `p=0.5` の決定境界を示す。

**ナレーション**

二次元のデータで、MAP 解のロジスティック回帰を見てみます。
背景の色は、クラス一である確率です。赤に近いほどクラス一、青に近いほどクラス二です。
白い線が、確率がちょうど 0.5 になる決定境界です。
MAP だけを見ると、重みは一つなので、各場所の確率も一つの値に決まります。
しかし、データが少ない領域では、この確率をそのまま信じすぎるのは危険です。

**原文参照**

PRML p.218 の 4.5.2 への導入では、ガウス近似した事後分布に関して予測時に周辺化する必要があることを述べている。MAP の一点予測は、この周辺化前の比較対象として扱う。

## Scene 5: 重みの不確実性はロジットの不確実性になる

**画面**

左に入力空間のデータと 2 つの評価点を表示する。右に `a=w^T phi` のガウス分布を 2 本表示し、データ近傍では狭く、離れた場所では広いことを示す。

**ナレーション**

新しい入力 phi に対して、ロジスティック回帰は a イコール w 転置 phi という一次元のロジットを作ります。
w の事後分布をガウスで近似しているので、a もまたガウス分布になります。
その平均は w MAP 転置 phi、分散は phi 転置 S N phi です。
データの近くでは、ロジットの分布は狭くなります。
データから離れるほど、同じ MAP 予測でも、ロジットの分布は広くなります。
予測では、このロジットの分布全体に対してシグモイドを平均します。

**原文参照**

PRML p.219 の Eq. (4.149) と Eq. (4.150) は、`a=w^T phi` の分布がガウスになり、平均 `mu_a=w_MAP^T phi`、分散 `sigma_a^2=phi^T S_N phi` で表されることを示す。

## Scene 6: 予測分布は確率を0.5側へ戻す

**画面**

左に MAP の確率面、右にラプラス近似後のベイズ予測確率面を並べる。上部に Eq. (4.151) と Eq. (4.153)-(4.154) の近似式を表示する。

**ナレーション**

予測したいクラス確率は、シグモイドに事後分布を掛けて、重み w について積分したものです。
ラプラス近似のもとでは、これは、シグモイドと一次元ガウスの積分になります。
PRML 4.5.2 では、この積分を、カッパを掛けた平均ロジットのシグモイドで近似します。
分散が大きいほどカッパは小さくなり、確率は 0.5 に近づきます。
一方で、確率 0.5 の境界は、平均ロジットがゼロの場所なので、MAP の境界と同じです。
変わるのは境界そのものではなく、境界から離れた確率の自信の持ち方です。

**原文参照**

PRML p.219 の Eq. (4.151) は、`int sigma(a) N(a|mu_a,sigma_a^2) da` を予測分布として書く。p.220 の Eq. (4.153)-(4.155) は、`kappa(sigma_a^2)=(1+pi sigma_a^2/8)^(-1/2)` を使い、`p(C1|phi,t)=sigma(kappa(sigma_a^2)mu_a)` と近似する。本文は、`p=0.5` の決定境界が MAP と同じになることも述べている。

## Scene 7: 4.5節の要点

**画面**

`p(w)`、`w_MAP`、`q(w)`、予測積分の 4 ステップを横に並べる。下に要点を箇条書きで表示する。

**ナレーション**

ベイズロジスティック回帰では、重みを事後分布として扱うことで、分類の不確実性を予測に反映します。
正確な事後分布と予測分布は扱いにくいため、PRML 4.5 ではラプラス近似を使います。
MAP を平均にし、ヘッセ行列の逆から共分散を作り、ガウス近似を得ます。
その上で、ロジット a の一次元ガウスに落とし込み、シグモイドとの積分を近似します。
この考え方は、ガウス過程分類や変分ロジスティック回帰にもつながる基本パターンです。

**原文参照**

PRML p.220 の 4.5.2 末尾は、ラプラス近似による予測分布の性質をまとめ、後の変分推論で同様の周辺化が示されることを述べている。関連する発展として Section 6.4.6 のガウス過程分類のラプラス近似、Section 10.6 の変分ロジスティック回帰がある。
