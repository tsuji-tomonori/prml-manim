# PRML 3.3 ベイズ線形回帰 ナレーション台本

## scene01 点推定から分布へ

- 三章三節では、線形回帰をベイズ的に見直します。
- これまでの最小二乗や正則化では、重みベクトルをひとつの値として求めました。
- ベイズ線形回帰では、重みそのものを確率分布として扱います。
- つまり、どの直線が正しいかを一つに決めるのではありません。
- データを見たあと、ありそうな直線の分布として答えるのが出発点です。

## scene02 事前分布と重み空間

- まず、データを見る前の重みの分布を置きます。
- ここでは切片と傾きの二つだけを考えると、重み空間は平面になります。
- 原点の近くを好むガウス分布を置くと、大きすぎる重みは最初から少し起こりにくくなります。
- この広がりを決めるのが、事前分布の精度アルファです。
- アルファが大きいほど、重みはゼロの近くに強く集まります。

## scene03 尤度が重みを絞る

- 観測点が一つ入ると、その点を説明しやすい重みに高い尤度が付きます。
- 重み空間で見ると、点一つは細長い帯のような制約になります。
- 事前分布に、この尤度を掛けて正規化すると、事後分布になります。
- 線形ガウスモデルでは、事後分布もガウス分布のままです。
- 形がガウスのまま更新できるので、平均と共分散だけで追跡できます。

## scene04 逐次更新

- データを一つずつ追加すると、事後分布は少しずつ狭くなります。
- 最初は多くの直線がありえますが、観測が増えるほど候補は絞られます。
- 重み空間の楕円が小さくなることと、グラフ上の直線のばらつきが小さくなることは同じ現象です。
- ここで大事なのは、一本の最良直線だけを追っていないことです。
- 不確実性がどのくらい残っているかも、学習結果の一部として残ります。

## scene05 予測分布

- 新しい入力エックススターでの予測も、ひとつの値ではなく分布になります。
- 平均は、事後平均の重みで作る通常の回帰曲線です。
- 分散には二つの成分があります。観測ノイズと、重みがまだ不確かなことによる揺らぎです。
- データが多い場所では帯が細く、離れた場所では帯が太くなります。
- ベイズ予測は、当たりそうな値と同時に、どのくらい自信があるかを返します。

## scene06 アルファとベータ

- 式に出てくるアルファとベータは、不確実性のつまみです。
- アルファは事前分布の精度で、重みをどれだけゼロ近くへ寄せるかを決めます。
- ベータは観測ノイズの精度で、データ点をどれだけ信用するかを決めます。
- ベータが大きいほどノイズは小さいとみなし、予測の帯は細くなります。
- この二つの値は、後のエビデンス近似でデータから選ぶ対象になります。

## scene07 3.4 への橋渡し

- ベイズ線形回帰は、点推定を分布推定へ広げました。
- 重みの不確実性を残すことで、予測分布と信頼の幅を自然に計算できます。
- さらに、分布全体を積分すると、モデルそのもののもっともらしさを比べられます。
- 次の三章四節では、この考えを使って、ベイズモデル比較へ進みます。
