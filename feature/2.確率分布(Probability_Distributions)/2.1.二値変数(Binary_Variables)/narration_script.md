# PRML 2.1 二値変数 ナレーション台本

対象: Christopher M. Bishop, *Pattern Recognition and Machine Learning* (2006), Section 2.1 Binary Variables

## scene01 二値変数とは

- 今回は、ピーアールエムエル二点一、二値変数を見ます。
- 二値変数とは、結果が二つだけの変数です。
- コインなら表か裏。検査なら陽性か陰性。クリックなら、した、しない。
- ここでは、結果を一なら成功、ゼロなら失敗、と書きます。
- たった二つの値ですが、この章の確率分布の基本形がここから始まります。

## scene02 Bernoulli 分布

- 二値変数エックスに対して、一が出る確率をミューと置きます。
- すると、ゼロが出る確率は、一マイナスミューです。
- この二つを一つの式にまとめたものが、ベルヌーイ分布です。
- エックスが一なら、式はミューになります。
- エックスがゼロなら、式は一マイナスミューになります。
- 画面のつまみでミューを動かすと、二本の棒の高さが反対向きに変わります。

## scene03 最尤推定

- 次に、データからミューを推定します。
- 十回観測して、一が七回、ゼロが三回出たとします。
- このデータが出やすくなるミューを選ぶのが、最尤推定です。
- ベルヌーイ分布では、尤度はミューの七乗かける、一マイナスミューの三乗になります。
- 曲線の山は、七割のところで最大になります。
- つまり最尤推定では、ミューは一の割合、エム割るエヌになります。

## scene04 少ないデータの不安定さ

- ただし、最尤推定だけを見ると、データが少ないときに強く言い切りすぎることがあります。
- 三回中三回が一なら、最尤推定はミューイコール一です。
- でも、たまたま三回続いただけかもしれません。
- ここで、ミューそのものにも確率分布を置きます。
- 二値変数のミューに対して便利なのが、ベータ分布です。
- ベータ分布は、ミューがゼロから一の間のどこにありそうかを表します。

## scene05 Beta 事前分布から事後分布へ

- ベータ分布には、エーとビーという二つのパラメータがあります。
- 直感的には、エーは一を見た分の重み、ビーはゼロを見た分の重みです。
- 観測で一がエム回、ゼロがエル回出ると、エーにはエムを足し、ビーにはエルを足します。
- これがベータ分布とベルヌーイ分布のきれいな関係です。
- 同じ形の分布のまま、パラメータだけが更新されます。
- この性質を、共役事前分布と呼びます。

## scene06 逐次更新

- 更新は、データをまとめて見なくても、一つずつ進められます。
- 一を見たらエーを一つ増やす。ゼロを見たらビーを一つ増やす。
- 曲線は、観測が増えるほど狭くなり、ミューのありそうな場所へ集まっていきます。
- 最尤推定は観測された割合だけを見ます。
- ベイズ推定では、事前の重みと観測データを足し合わせて、分布全体を更新します。

## scene07 予測分布と次節への橋渡し

- 最後に、次の一回が一になる確率を考えます。
- ベイズ的な予測では、ミューの一点だけではなく、事後分布全体を平均します。
- 結果は、エムプラスエーを、エヌプラスエー足すビーで割った形になります。
- これは、観測された一の数に、事前分布の重みを足した割合です。
- 二値変数では、一かゼロの二択でした。
- 次の二点二では、三択以上の多項変数へ広げます。
